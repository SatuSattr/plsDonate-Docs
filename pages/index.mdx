# Welcome to plsDonate Docs

Welcome to the official documentation for **plsDonate**!

plsDonate is a PaperMC plugin that allows server administrators to receive donations and players to donate real money directly via the game.

## Overview

This documentation will help you get started with setting up and configuring plsDonate on your server.

- **plsDonate Express**: The lite version, currently available.
- **plsDonate Full**: Coming soon!

Select a topic from the sidebar to get started.
