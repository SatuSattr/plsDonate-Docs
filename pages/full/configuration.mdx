# Configuration

import { Callout } from 'nextra/components'

<Callout type="warning">
  plsDonate Full is currently under development. Configuration details will be updated upon release.
</Callout>

After installing plsDonate and starting your server, a `config.yml` file will be generated at:

```
/plugins/plsDonate/config.yml
```

## config.yml

```yaml
# plsDonate Configuration

# General Settings
plugin:
  prefix: "&c[plsDonate]&r"
  language: "en"

# Database Settings
database:
  type: "mysql" # sqlite or mysql
  host: "localhost"
  port: 3306
  name: "plsdonate"
  username: "root"
  password: ""

# Donation Settings
donation:
  currency: "USD"
  minimum: 1.00
  reward-tiers: true

# Payment Gateways
gateways:
  enabled: []
  # - paypal
  # - stripe
```
