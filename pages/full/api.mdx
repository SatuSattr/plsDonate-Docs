## Events

| Event | Description |
|---|---|
| `DonationReceivedEvent` | Fired when a donation is successfully processed |
| `DonationCommandEvent` | Fired when a player uses the `/donate` command |
| `RewardTierReachedEvent` | Fired when a player reaches a donation reward tier |
| `GatewayResponseEvent` | Fired when a payment gateway returns a response |

## Usage Example

```java
@EventHandler
public void onDonation(DonationReceivedEvent event) {
    Player player = event.getPlayer();
    double amount = event.getAmount();
    String tier = event.getRewardTier(); // null if no tier reached
    player.sendMessage("Thanks for donating $" + amount + "!");
}
```
