# Commands & Permissions

## Player Commands

| Command | Description | Permission |
|---|---|---|
| `/donate <amount>` | Donate to the server | `plsdonate.donate` |
| `/donate help` | Show help menu | `plsdonate.donate` |

## Admin Commands

| Command | Description | Permission |
|---|---|---|
| `/plsdonateadmin reload` | Reload the plugin config | `plsdonate.admin.reload` |
| `/plsdonateadmin list` | View recent donations | `plsdonate.admin.list` |

## Permissions

| Permission | Description | Default |
|---|---|---|
| `plsdonate.donate` | Allow player to use donate command | `true` |
| `plsdonate.admin` | Access to all admin commands | `op` |
| `plsdonate.admin.reload` | Reload plugin config | `op` |
| `plsdonate.admin.list` | View donation history | `op` |
