# plsDonate Express

import { Callout } from 'nextra/components'

<Callout type="info">
  **plsDonate Express** is the lite version of plsDonate — available now and ready to use on your server!
</Callout>

**plsDonate Express** is a lightweight PaperMC plugin that enables server administrators to receive real money donations from players directly in-game, with a simplified setup and core donation features.

## What's Included

- In-game donation commands for players
- Admin panel for managing donations
- Basic notification system
- Lightweight & easy to configure
