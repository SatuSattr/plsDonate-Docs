# plsDonate Documentation

Official documentation for the **plsDonate** Minecraft PaperMC plugin suite. This website provides detailed guides for both **plsDonate Express** (Lite) and the upcoming **plsDonate Full** version.

![plsDonate Banner](./public/banner.png)

## 🚀 Features

- **Built with Nextra**: Powered by Next.js and Nextra for a seamless documentation experience.
- **Modern UI/UX**: Custom themed with the plsDonate brand colors (Red & Gold) and a professional sharp-edged design.
- **Dual Product Support**: Integrated documentation for both Express and Full versions.
- **Dark Mode**: Supports system, light, and dark themes with custom Lucide icons.
- **SEO Optimized**: Includes Open Graph and Twitter Card meta tags for rich social media previews.
- **Fast Navigation**: Includes a YouTube-style NProgress loading bar for smooth page transitions.

## 🛠️ Getting Started

### Prerequisites

- [Node.js](https://nodejs.org/) (v18 or higher recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/SatuSattr/plsDonate-Docs.git
   cd plsDonate-Docs
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) in your browser.

## 📁 Project Structure

- `/pages`: MDX/JSX files for all documentation pages.
- `/public`: Static assets (logos, banners).
- `/styles`: Global CSS and custom design tokens.
- `theme.config.jsx`: Nextra theme configuration and custom navbar logic.

## 🚢 Deployment

This project is optimized for deployment on [Vercel](https://vercel.com):

1. Connect your GitHub repository to Vercel.
2. Vercel will automatically detect the Next.js project.
3. Deploy with a single click.
